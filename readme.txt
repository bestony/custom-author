帮助你在发布文章时自定义作者信息

== Description ==

帮助你在发布文章时自定义作者信息，免于手动新建作者。

== Installation ==

1. 从 WordPress 后台的「插件」-「安装插件」页面，上传插件。
2. 发布文章时在发布内设置作者名。
